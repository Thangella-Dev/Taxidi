# 13 - Taxiro Go-To-Market Strategy

**Date:** 23 July 2026

## GTM Objective

Launch Taxiro in a controlled local market, prove ride completion, safety, rider earnings, payment reliability, and repeat usage before expanding.

## Launch Principle

Do not start broad. Start in one zone and make that zone excellent.

## Phase 1 - Pre-Launch Validation: 2-4 Weeks

### Goals

- Identify exact launch area.
- Understand local regulations.
- Recruit early riders/drivers.
- Test user interest.
- Prepare payment/support/safety operations.

### Actions

- Choose one city zone with high short-distance demand.
- Speak to 50-100 potential users.
- Speak to 30-50 potential riders/drivers.
- Check transport department rules for bike/auto/cab aggregation.
- Identify launch corridors: hostels, colleges, offices, apartments, metro/bus points.
- Create WhatsApp/Telegram early-access community.
- Prepare early rider onboarding and verification process.

### Cost Estimate

|---|---:|
| Local travel/research | Rs 5,000-Rs 20,000 |
| Survey/forms/printing | Rs 1,000-Rs 5,000 |
| Early landing page/social creatives | Rs 0-Rs 10,000 |
| Legal/regulatory consultation | Rs 10,000-Rs 50,000 |
| **Total** | **Rs 16,000-Rs 85,000** |

## Phase 2 - Closed Pilot: 4-8 Weeks

### Goals

- Run real rides with limited users/riders.
- Validate fare, pickup, tracking, cancellation, support, and payment flow.
- Measure repeat usage.

### Target

- 50-100 riders/drivers.
- 500-2,000 users.
- 1-3 launch zones.
- 1,000-5,000 completed rides.

### Actions

- Onboard verified riders manually.
- Give riders training and app walkthrough.
- Launch invite-only user access.
- Offer limited first-ride coupons.
- Monitor every failed/cancelled ride.
- Maintain manual support over WhatsApp/calls initially.
- Track daily supply/demand heatmap.

| Item | Monthly Cost |
|---|---:|

| User coupons/referrals | Rs 25,000-Rs 1,00,000 |

| API/cloud/payment/SMS | Rs 15,000-Rs 75,000 |
| Support operations | Rs 10,000-Rs 50,000 |
| **Total/month** | **Rs 95,000-Rs 4,55,000** |

## Phase 3 - Public Local Launch: 2-3 Months

### Goals

- Open app to public users in selected city/zone.
- Improve brand visibility.
- Build repeat user base.
- Improve rider earnings and supply reliability.

### Actions

- Launch Play Store app.
- Run local Instagram/Google ads.
- Partner with hostels, PGs, colleges, offices, cafes, gyms.
- Add referral program.
- Add rider earning guarantee for peak hours.
- Add local support number.
- Add safety messaging.

### Cost Estimate

## Launch Metrics

Track these daily:

- New users.
- Active users.
- New riders.
- Online riders by hour.
- Ride requests.
- Accepted rides.
- Completed rides.
- Cancellation rate.
- Average pickup time.
- Average fare.
- Platform commission.
- Incentive spend.
- Support tickets.
- Repeat user rate.
- Rider earnings.

## Expansion Rule

Do not expand to a new zone until current zone reaches:

- 80%+ ride acceptance rate.
- Less than 10-15% cancellation rate.
- Average pickup time below 8-10 minutes.
- Positive or controlled contribution margin.
- Reliable support handling.
