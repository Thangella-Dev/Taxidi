# 04 - Missing Real-World Features

**Date:** 23 July 2026

Taxiro has strong MVP coverage, but the following are still needed before it can be considered a real commercial ride-hailing platform.

## Payment And Wallet

- Real payment gateway integration.
- User wallet top-up.
- Wallet ledger credit/debit/refund.
- Gateway webhook verification.
- Refund processing.
- Failed payment recovery.
- Driver payout settlement.
- Payment reconciliation dashboard.
- GST invoice/receipt support.

## Notifications

- Real push notifications through Firebase Cloud Messaging.
- SMS/OTP provider.
- WhatsApp emergency/support alerts.
- Notification delivery status.
- Critical alert escalation.

## Maps And Tracking

- Production-grade maps provider or self-hosted OSM stack.
- Production geocoding/search.
- Traffic-aware ETA.
- Snap-to-road tracking.
- Native mobile background rider tracking.
- Geofencing and route deviation alerts.

## Rider Verification / KYC

- Aadhaar/PAN/driving licence/RC/insurance verification.
- Liveness/selfie verification provider.
- Admin verification audit trail.
- Document expiry reminders.

## Business Operations

- Driver payout dashboard.
- Refund/dispute management.
- Customer support SLA system.
- Call masking.
- Coupon/referral campaign analytics.
- Finance exports.
- City/service zone rollout control.

## Security / Compliance

- Full RLS security audit.
- Rate limiting.
- Admin audit logs review.
- Privacy/legal review.
- Payment compliance review.
- Penetration testing.
- Load testing.
